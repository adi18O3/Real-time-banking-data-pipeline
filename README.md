# 🏦 Big Data Bank Customer Segmentation Project

## 🚀 End-to-End Big Data + ML Pipeline

### Architecture
Data → S3 → PySpark → Preprocessing → ML Models  
                        ↓  
                       Hive → Tableau Dashboard  

## 📊 ML Tasks
- Customer Segmentation (KMeans)
- High-Value Customer Prediction (Random Forest)

## ▶ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Configure AWS:
   aws configure

3. Run full pipeline:
   python src/main.py

## 📌 Business Impact
- Identify high-value customers
- Segment customers for marketing
- Predict premium customer potential
